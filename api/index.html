Bkio device resource
major
Integer The major number of the device. format: uint64
minor
Integer The minor number of the device. format: uint64
weight (optional)
Integer The weight of the device against the descendant nodes. format: uint16
leaf_weight (optional)
Integer The weight of the device against the sibling nodes. format: uint16

BlkIoDeviceThrottleResource - Up

Provides the ability to throttle a device (both byte/sec, and IO op/s)
major
Integer The major number of the device. format: uint64
minor
Integer The minor number of the device. format: uint64
rate
Integer The rate. format: uint64

BlkIoResources - Up

Blkio controller
weight (optional)
Integer The weight of the control group against descendant nodes. format: uint16
leaf_weight (optional)
Integer The weight of the control group against sibling nodes. format: uint16
weight_device
array[BlkIoDeviceResource] For each device, a separate weight (both normal and leaf) can be provided.
throttle_read_bps_device
array[BlkIoDeviceThrottleResource] Throttled read bytes/second can be provided for each device.
throttle_read_iops_device
array[BlkIoDeviceThrottleResource] Throttled read IO operations per second can be provided for each device.
throttle_write_bps_device
array[BlkIoDeviceThrottleResource] Throttled written bytes/second can be provided for each device.
throttle_write_iops_device
array[BlkIoDeviceThrottleResource] Throttled write IO operations per second can be provided for each device.

CpuResources - Up

Cpu controller
cpus (optional)
String A comma-separated list of CPU IDs where the task in the control group can run. Dashes between numbers indicate ranges.
mems (optional)
String Same syntax as the cpus field of this structure, but applies to memory nodes instead of processors.
shares (optional)
Integer Weight of how much of the total CPU time should this control group get. Note that this is hierarchical, so this is weighted against the siblings of this control group. format: uint64
quota (optional)
Long In one period, how much can the tasks run in nanoseconds. format: int64
period (optional)
Integer Period of time in nanoseconds. format: uint64
realtime_runtime (optional)
Long This is currently a no-operation. format: int64
realtime_period (optional)
Integer This is currently a no-operation. format: uint64

MemoryResources - Up

Memory controller
kernel_memory_limit (optional)
Long How much memory (in bytes) can the kernel consume. format: int64
memory_hard_limit (optional)
Long Upper limit of memory usage of the control group's tasks. format: int64
memory_soft_limit (optional)
Long How much memory the tasks in the control group can use when the system is under memory pressure. format: int64
kernel_tcp_memory_limit (optional)
Long How much of the kernel's memory (in bytes) can be used for TCP-related buffers. format: int64
memory_swap_limit (optional)
Long How much memory and swap together can the tasks in the control group use. format: int64
swappiness (optional)
Integer

Controls the tendency of the kernel to swap out parts of the address space of the tasks to disk. Lower value implies less likely.

Note, however, that a value of zero does not mean the process is never swapped out. Use the traditional mlock(2) system call for that purpose.

format: uint64
